# Electromart

Electromart is a MERN-style ecommerce application with a React storefront, Express API, MongoDB models, admin surfaces, and M-Pesa payment integration.

## Production Readiness

The production hardening plan lives in [docs/production-readiness-workflow.md](docs/production-readiness-workflow.md).

## Common Commands

```bash
npm install
npm run lint
npm test
npm run build
npm run verify
```

Use `.env.example` as the template for local and deployment environment variables. Keep real `.env` files out of git.
